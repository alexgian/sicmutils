SICMUTILS Reference Manual

This is a description of the SICMutils computer algebra system, a port to Clojure of scmutils, the MIT-Scheme system devised and used by Gerry Sussman and others at MIT, to teach advanced mechanics from a computing perspective.  The landmark book which introduced these ideas is Structure and Interpretation of Classical Mechanics (aka SICM), and can be seen as the evolution into physics of the era-defining ideas first presented in Structure and Interpretation of Computer Programs (aka SICP).

Following the scmutils paradigm, SICMutils is an integrated library of procedures, embedded in the programming language Clojure,  intended to support teaching and research in mathematical physics and electrical engineering. SICMutils and Clojure are particularly effective in work where the functional nature of Clojure is advantageous (such as classical mechanics, among others) where many of the procedures are most easily formulated as quite abstract manipulations of functions.

One issue that was holding back the uptake of this system was its reliance on a non-portable feature of MIT-Scheme, for which Clojure was able to provide a very capable alternative. This means that SICMutils is now available in a modern, Java-based environment, and thus the whole arsenal of Java utilities will be easily available to user. We hope that this will prove beneficial and allow the harnessing of many already existent tools - especially in visualization - to the educational process at hand.

The scmutils implementation is vast, and at this stage only a small subset of it has been ported to SICMutils.  The main driving motivation was to get the all code in the SICM book up and running, so that it could be used by students.  As more functionality is added, SICM should become a fairly complete elegant computer algebra system (CAS).  It also has the advantage that it is free software, whereas most of the competing heavyweight packages are quite proprietary.

This particular document has followed the form of the original scmutils Reference Manual quite closely.   Even though a lot of changes have been made of necessity, either due to language differences or to incompleteness, the chapter structure has been retained.  The original manual can be found here.  The full scmutils distribution can be found here.

Many people contributed to the development of the scmutils library over many years and you can find their names on the original manual already mentioned. SICMutils is due to the work and inspiration of Colin Smith. It is available via GitHub at [littleredcomputer/sicmutils].


Please note that if you have been already using scmutils on MIT-Scheme, you will find that there are some small but significant differences with this system – due to language idiosyncrasies - which could lead to potential ‘gotchas’. Reading through this document should make those differences clear.
