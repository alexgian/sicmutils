SICMutils Numbers

Since Scheme and Clojure implement their numeric systems differently and a number of these differences become apparent when comparing scmutils to the SICMutils system.  It is, however, not too difficult to get the two systems to behave in a satisfactorily similar manner.

The Clojure number system is based on the underlying Java implementation.  As such it provides machine based representations of numbers viz. Integers (Long), and floating point decimals (Double).  These have been augmented in SICMmutils by Big Integers (unlimited size), fractions and complex numbers.

Applying the Clojure procedure type to various numbers gives the following results:

(type -3)      ==> java.lang.Long

(type 3.0)     ==> java.lang.Double

(type 2/3)     ==> clojure.lang.Ratio

(type 30000000000000000000)   ==> clojure.lang.BigInt

(type (complex 1 1))          ==> org.apache.commons.math3.complex.Complex

The above (with the exception of complex numbers) will return true when given as the argument to the function

(number? x)

Complex numbers are a recent addition to the system and are based on the Apache Commons Math library.  In are some  respects they have not yet been integrated into SICMutils, so for now, number? must mean Real number.

Note that the infinite precision BigInt type is indicated in  indicated in SICMutils by a suffix "N".  Exact arithmetic, performed with fractions, uses this type, so a number does not need to be large in order to be a BigInt:

(* 2/3 12/4)  ==>  2N

The system also provides a function exact? which will return true only if the number is exact, i.e. integer or ratio but not floating point approximation.

Other numerical functions are much as one would expect, here is a brief list:

(zero? n)     true if the value of n = 0
(negate n)    multiply n by -1
(abs n)       the magnitude of n
(invert n)    1 divided by n
(square n)    multiply n by itself
(cube n)      the square times the number
(expt n m)    n raised to the power of m
(sqrt n)      the square root of n

For Integers

(gcd n m)     the greatest common divisor of n and m

and the usual trigonometric and loarithimic functions

(exp x)  e^x
(log x)  natural log of x
(sin x)  and  (asin x)
(cos x)  and  (asin x)
(tan x)  and  (atan x)

Further, complex numbers support the functions

(real-part c)  the real part
(imag-part c)  the imaginary part
(magnitude c)  the absolute value (by Pythagoras)
(abs c)        same as above
(angle c)      the Argument of a complex number
(conjugate c)  the number with imag-part negated
